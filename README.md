# Shortly — Link Shortener Landing Page

A modern landing page built with React, TypeScript, and Vite for **Shortly**, a URL shortening service. The page showcases the product's value proposition with a hero section, feature highlights (Brand Recognition, Detailed Records, Fully Customizable, Analytics), a call-to-action section, and a footer.
